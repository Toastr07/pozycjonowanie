Pozycjonowanie elementów w CSS

Przegląd właściwości position

1. Właściwość position

Właściwość position wpływa na ułożenie elementów na stronie internetowej.

Może posiadać wartość:

  • static
  • relative
  • fixed
  • absolute
  • sticky

2. Wartość static

Wartość static jest domyślną wartością właściwości position.

Element pod wpływem tej wartości jest upozycjnowany normalnie.

3. Wartość relative

Element pod wpływem tej wartości jest upozycjonowany w dowolnym miejscu na stronie za pomocą właściwości:

  • left
  • top
  • right
  • bottom

Element jest traktowany jakby dalej był w pierwotnym miejscu. Jest jedynie wyświetlany w innej pozycji.

4. Wartość fixed

Element pod wpływem tej wartości jest upozycjonowany w dowolnym miejsu względem okna przeglądarki. Aby ustawić jego pozycje używamy właściwości:

  • left
  • top
  • right
  • bottom

Element jest traktowany jakby nie zajmował miejsca na stronie

5. Wartość absolute

Element pod wpływem tej wartości jest upozycjonowany w dowolnym miejsu względem przodka. Aby ustawić jego pozycje używamy właściwości:

  • left
  • top
  • right
  • bottom

Element jest traktowany jakby nie zajmował miejsca na stronie

6. Wartość sticky

Tak jak w przypadku wartości relative, element pod wpływem tej wartości jest upozycjonowany w dowolnym miejscu na stronie za pomocą właściwości:

  • left
  • top
  • right
  • bottom

Gdy element normalnie wyjechałby poza okno (podczas scrollowania), element "przyklei" się do okna i będzie przewijał razem z nim (jak w przypadku wartości fixed).

Element jest traktowany jakby dalej był w pierwotnym miejscu. Jest jedynie wyświetlany w innej pozycji.
